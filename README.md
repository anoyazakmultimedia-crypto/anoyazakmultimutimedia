Project Overview
This is a professional portfolio website for Anoya Zak Multimedia, a creative services business specializing in multimedia production, digital design, and crypto‑related educational content. The website showcases their services including video editing, photography, graphic design, crypto awareness resources, and digital branding solutions.


Website Structure
The website consists of 5 interconnected pages:

index.html – Home page with hero section and service highlights

services.html – Detailed description of multimedia and crypto services offered

about.html – Company profile, mission, vision, and values

contact.html – Contact form, business information, and social media links


account.html- create sign up to portal if first time user , signin, Username/email field, Password field, “Forgot password” link.

Setup Instructions
Step 1: Create Folder Structure
Create a main folder named anoya-zak-multimedia

Inside it, create three folders: css, js, images

Step 2: Add Files
Place all HTML files in the main folder

Place styles.css in the css folder

Place script.js in the js folder

Step 3: Add Images
Logo: images/logo.png (200x80 pixels recommended)

Hero background: images/hero-bg.jpg (1920x600 pixels recommended)

Gallery images: images/gallery/

Service images: images/services/

Step 4: Test Locally
Open index.html in a browser

Navigate through all pages

Test mobile responsiveness, contact form, and gallery filters

GitHub Hosting Instructions
Create a GitHub account (if not already).

Create a repository named anoya-zak-multimedia.

Upload all files and folders.

Enable GitHub Pages under repository settings.

Access your live site at:https://YOUR-USERNAME.github.io/anoya-zak-multimedia/

Troubleshooting
Links not working? Check file names and paths.

CSS not loading? Verify <link rel="stylesheet" href="css/styles.css">.

JavaScript errors? Check console logs (F12).

Images missing? Confirm correct paths and file extensions.
